Description of the input form parameters

Soil Type

The soil characteristics or quality of soil, there are 16 options to specify the soil category corresponding to your field/farm.
However, if you dont know the type of soil you have, please use the http://www.info-sols.ca service to determine soil characteristics.

Previous Crop

Previously harvested crop at the given farm, this will be used to determine the nutrient contribution of fertilizer application.
The previous crop to previous nutrient contribution will be used as one of the factor in estimating the results.

Tillage System

The tillage system followed

Soil Organic Matter SOM (%)

Precipitation (AWDR, mm)

The rain amount data based on the concept of abundant and well-distributed rainfall (AWDR) proposed by Tremblay et al. (2012). The AWDR was categorized into 5 high-level subjective classes for user to specify the rain in the input form. The user can select one of the class values based on the expected rainfall for the growing season in the specified region

Temperature (CHU, °C)

Temperature (CHU, °C): The corn heat unit (CHU) expresses temperature measurements in degrees (Bootsma et al., 2005). This temporal factor can affect the yield growth/production;therefore, the user needs to specify suitable conditions pertaining to his/her site. It was categorized into five high-level classes according to the CHU values
